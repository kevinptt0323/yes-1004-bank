## Development

### Required

- php 5.6.6
- MySQL 5.6.14
- Node.js 0.10.33
- [npm](https://github.com/npm/npm/)

### Development

第一次請先執行

	$ npm install
	$ bower install

發布時請執行

	$ gulp
